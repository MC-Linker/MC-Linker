Lass dir Statistiken, Erfolge und Inventare deines Minecraft Servers innerhalb von Discord anzeigen, führe Befehle aus und erstelle eine Chat-Brücke mit MC Linker!

## Beschreibung

Moderiere und verbinde deinen Minecraft-Server mit Discord! Dieser Bot erlaubt dir, Minecraft-Statistiken, Fortschritte und sogar Inventare von jedem Mitglied in Discord anzuzeigen. Unterstützt auch einen umfangreichen Zwei-Wege-Chat mit Minecraft, Befehlsausführung und viele andere hilfreiche Funktionen.

## Einrichtung

Um die meisten Befehle nutzen zu können, musst du deinen Minecraft Java-Edition Server mit dem Bot verbinden. Dies kann durch die folgenden Schritte erledigt werden:

+ Füge das "Discord-Linker" Plugin zu deinem Minecraft-Server von [SpigotMC](https://www.spigotmc.org/resources/discord-linker.80691/), [Modrinth](https://modrinth.com/plugin/discord-linker), die [PaperHangar](https://hangar.papermc.io/Lianecx/Discord-Linker) oder die [Aternos Plugin Liste](https://aternos.org/addons/a/spigot/98749) hinzu
+ Starte deinen Minecraft-Server neu oder führe `/reload confirm` auf deinem Minecraft-Server aus
+ Führe `/connect` in Discord aus
+ Folge den Anweisungen, die du per persönliche Nachricht in Discord erhältst

## Wichtige Befehle

+ `/help`: Detaillierte Beschreibung und Verwendung jedes Befehls
+ `/inventory`: Zeige das Inventar eines Mitglieds an
+ `/chatchannel`: Lege einen Kanal fest, in dem der Bot den Minecraft-Chat sendet (nur für Plugin)
+ `/statchannel`: Legen Sie einen Kanal fest, in dem der aktuellen Status oder die Anzahl der Spieler anzeigt wird
+ `/command`: Führe jeden Minecraft-Befehl aus
+ `/stats`: Zeige Minecraft-Statistiken von einem Spielers an
+ `/advancements`: Zeige Minecraft-Fortschritte von einem Spieler an
+ `/connect account`: Verbinde dein Discord-Account mit deinem Minecraft-Account
+ `/connect`: Verbinde deinen Minecraft-Server mit dem Bot
+ `/ban`: Banne einen Spieler direkt vom Minecraft-Server
+ `/serverinfo` oder `/userinfo`: Zeigt Informationen über Ihren Server oder einen online Spieler an
